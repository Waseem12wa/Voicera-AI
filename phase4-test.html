🛠️ Manual Setup Instructions

Step 1: Install Prerequisites
1. Install MongoDB: https://www.mongodb.com/try/download/community
2. Install Redis: https://redis.io/download (or use Docker)
3. Ensure Node.js 18+ is installed
Step 2: Start Services
1. Open 4 separate terminal windows
2. Run each service:
  • Voice Service: cd microservices/voice-service && npm start
  • User Service: cd microservices/user-service && npm start
  • Analytics Service: cd microservices/analytics-service && npm start
  • API Gateway: cd microservices/api-gateway && npm start
Step 3: Test Services
Use the test buttons above or run:
curl http://localhost:3000/health
curl http://localhost:3001/health
curl http://localhost:3002/health
curl http://localhost:3003/health
